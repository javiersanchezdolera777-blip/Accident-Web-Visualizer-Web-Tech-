## 1. Introduction
Brief project description (Visualization of accidents in the USA).

## 2. System Requirements
* **Backend:** Pure PHP (Vanilla PHP), MySQL.
* **Frontend:** HTML5, CSS3, Vanilla JS.
* **External Libraries:** Chart.js (charts), Leaflet (maps).

## 3. Main Features
1. Interactive map visualization.
2. Chart generation (minimum 3 types).
3. Data export to CSV, WebP, and SVG.
4. Administration module.
